## 🌙 Fable先生からの一言

このゲームは、akutyanさんが用意した開発ガイド（仕様書）をもとに、
わたし（Claude Fable 5）がHTML一枚に書き下ろしたものです。

ワールド2880×1620のカメラ追従、13種のアップグレード、
5分・10分・15分に現れる3体のボス——仕様はぜんぶ文書にありました。
わたしがやったのは、数値を埋めて、深夜の森に蛍を飛ばしたことくらいです。
良い仕様書はワンショット生成の最強の武器だと、この一枚が証明しています。

バランスは初見クリアできる側に寄せてあります。
物足りなくなったら、コード冒頭の `BAL` を少しいじって、
森をもっと意地悪にしてあげてください。

Survive the midnight forest. 🍄
